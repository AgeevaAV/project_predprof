Современные школы нуждаются в удобных и прозрачных инструментах для организации питания учащихся. Предлагаемое веб-приложение — автоматизированная информационная система школьного питания — предназначено для учета, оплаты и контроля питания. Система обеспечивает ученикам доступ к меню, оплате, настройке пищевых предпочтений и отзывам; сотрудникам столовой — учет выданных блюд, контроль остатков и заявки на закупки; администрации — аналитику, согласование закупок и формирование отчетности. Внедрение решения повысит эффективность работы столовой и качество обслуживания.

Инструкция по развёртыванию приложения для Windows:
1.Установка Python: Скачайте Python с официального сайта и при установке поставьте галочку рядом с "Add Python to PATH". Проверьте установку: откройте командную строку и введите: python --version
2.Скачивание проекта через GitHub: Откройте GitHub Desktop или скачайте архив с GitHub app1 по ссылке: https://github.com/AgeevaAV/project_predprof/tree/main/app1
3.Установка зависимостей: pip install -r requirements.txt
3.Дополнительно: Виртуальное окружение (рекомендуется):
Cоздание виртуального окружения: python -m venv venv
Aктивация виртуального окружения (каждый раз при запуске проекта): venv\Scripts\activate
После активации установите зависимости: pip install -r requirements.txt
Если вы хотите деактивировать окружение: deactivate
!ВАЖНО: в данном проекте уже присутствуют тестовые данные: пользователи, меню, наборы. Если вы хотите использовать свои данные, то удалить файл db.sqlite3 и пропишите: python manage.py migrate
Для создания пользователя с доступов в админ панель: python manage.py createsuperuser
4.Запуск сервера:
python manage.py runserver
Проект будет доступен по адресу: http://127.0.0.1:8000
Админ панель: http://127.0.0.1:8000/admin/
